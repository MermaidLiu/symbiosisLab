# Symbiosis Lab 产品使用手册

> 实验室资源管理平台 · 仪器预约 / 动物代管 / 科研助理工作台  
> 版本对应仓库主分支（`main`）当前功能

---

## 1. 产品简介

Symbiosis Lab 面向高校/实验室场景，统一管理：

- **仪器**：浏览、预约、审批；仪器管理员可导入/维护仪器
- **实验动物**：笼位总览、代管动物、申领审批、兽医处理
- **科研助理**：项目全流程事项板、荣誉库、照片库、PPT 制作

登录后首页（工作台）会按角色自动切换布局；侧栏菜单也会按权限显示。

界面支持 **中文 / English** 切换（右上角语言）。

## 4. 通用功能

### 4.1 通知

- 右上角铃铛查看待办通知（如预约待批、申请状态变更）  
- 可跳转至对应业务页处理  

### 4.2 我的预约（`/bookings`）

- 查看本人提交的仪器/资源预约  
- **仪器负责人 / 管理员**可在此 **批准 / 拒绝** 待审预约  

### 4.3 操作申请（`/animals/applications`）

- 学生：只看 **自己的** 申请，可取消待审申请  
- 管理员 / 动物负责人 / 兽医（按类型）：可审核  

申请类型包括：代管申领、兽医、转移、笼位变更、繁殖、安乐死等。

### 4.4 提交周报（`/progress`）

- 普通用户可见「提交周报」入口  
- 科研助理账号侧栏不显示该入口（由助理侧收集进度）

## 5. 学生 / 普通用户

### 5.1 工作台

首页可查看：

- 待审 / 已批预约数量  
- 待审申领、已申领小鼠、可申领数  
- 快捷入口：预约仪器、申领动物、我的预约、我的申请  
- 「我的预约」「我的申领」「已申领小鼠」列表  

### 5.2 预约仪器

1. 打开 **仪器管理**（`/instruments`）  
2. 点击仪器进入详情  
3. 在周历上拖选时段（支持约 **0.5–24 小时**，以该仪器规则为准）  
4. 填写实验目的并提交  

也可在仪器列表页使用 **导入 CSV** 批量登记仪器（模板见 [仪器批量导入模板.csv](./import-templates/仪器批量导入模板.csv)）。  

**注意：**

- 仪器 **维护中 / 已退役**：不可预约；维护中可悬停状态查看预计修好时间  
- 仪器勾选 **需培训** 且本人未完成培训：**不可选时段**  
- 预约成功后进入待审批，需仪器负责人批准  

### 5.3 申领动物

1. 打开 **代管动物**（`/animals/managed`）  
2. 页面为 **可申领目录**：已过滤空白鼠、已有申领人、待审中、死亡个体  
3. 勾选小鼠 → **申领所选**（或在详情中申领）  
4. 提交后生成 **代管申请**，等待管理员审核  
5. 通过后，小鼠会出现在工作台「已申领小鼠」  

也可使用 **批量上传**：按 [小鼠模板](./import-templates/小鼠批量导入模板.csv) 整理 CSV 后粘贴导入。用途非空白且申领人为「未分配」时，自动归到本人名下。  

**规则：** 空白用途小鼠不可申领；已被他人申领或已有待审申领的不可重复提交。

### 5.4 其他申请

在 **操作申请** 中可新建兽医、转移等非申领类申请（按业务需要填写说明）。

## 6. 动物房主管

### 6.1 笼位总览工作台（首页 / `/animals/facility-board`）

- **统计**：笼位与小鼠相关汇总  
- **每日待办**：勾选完成、新增待办  
- **工作日历**：圆圈标今天；有记录的日期有小点；点击查看当日工作流  
- **笼位板**：按用途颜色区分；可新增笼位、批量上传等（视权限）  

### 6.2 代管动物

- Excel 风格列表：用途、电生理、入笼、植入、采集、跟踪、处死方式、特殊实验、状态等  
- 可查看详情、记录信号、强制处死等管理操作  
- 审批学生的代管申领（在操作申请中）  
- **批量导入小鼠**：笼位总览 → 批量上传；模板见 [小鼠批量导入模板.csv](./import-templates/小鼠批量导入模板.csv)，说明见 [import-templates/README.md](./import-templates/README.md)

## 8. 仪器负责人

### 8.1 管理工作台

- 我的仪器数量、维护中数量、待审预约  
- 快捷进入仪器库与预约审批  

### 8.2 仪器库（`/instruments`）

**浏览（所有人）**

- 卡片展示图片、状态、型号、位置、审批联系人、是否需培训、预约时长规则  
- **维护中** 状态悬停可看：预计修好时间、备注、维修对接人  

**管理（仪器负责人）**

| 能力 | 说明 |
|------|------|
| 添加仪器 | 填写中英文名、型号、位置、状态、培训要求、0.5–24h 时长、四步联系人、配件、图片等 |
| 编辑仪器 | 详情页「编辑仪器」 |
| 导入 CSV | 批量导入字段；图片可事后在编辑中上传 |
| 上传图片 | 支持常见图片格式 |
| 登记培训 | 对「需培训」仪器，在详情页为用户标记培训完成 |
| 预约历史 | 查看并导出该仪器预约记录 |

**四步联系人**

1. 预约审批  
2. 使用培训  
3. 日常运维  
4. 维修对接  

### 8.3 审批预约

在 **我的预约**（`/bookings`）中批准或拒绝待审预约。

### 8.4 CSV 导入列参考

模板文件（推荐直接下载填写）：

- [仪器批量导入模板.csv](./import-templates/仪器批量导入模板.csv)
- 字段说明：[import-templates/README.md](./import-templates/README.md)

常用列名（中英均可）：`name` / `名称`，`nameEn`，`model`，`location`，`status`，`contactPhone`，`trainingRequired`，`minBookingHours`，`maxBookingHours`，`maintenanceUntil`，以及 `approvalName`/`approvalPhone`、`trainingName`/`trainingPhone`、`operationsName`/`operationsPhone`、`repairName`/`repairPhone` 等。

导入弹窗内也有示例表格，可直接修改后导入。

## 9. 科研助理

### 9.1 工作台（首页）

- 顶部统计快捷卡片（申报进行中、经费、照片库、荣誉库等）  
- **每日待办** + **工作日历**（有截止/更新的日期有小点；点击查看当日事项）  
- 侧栏进入各业务模块  

### 9.2 七大职能事项板

每个模块均为「事项 + 状态流 + 标准工作清单」：

| 菜单 | 典型流程 |
|------|----------|
| 项目申报 | 组织申报 → 形式审查 → 已报送 → 跟踪中 → 完成 |
| 过程管理 | 立项手续 → 过程监控 → 进展跟踪 → 协调解决 → 完成 |
| 结题验收 | 材料准备 → 组织验收 → 登记归档 → 完成 |
| 经费监管 | 预算编制 → 使用监控 → 合规初审 → 决算审计 → 完成 |
| 信息维护 | 待录入 → 已更新 → 已核对 → 完成 |
| 内外协调 | 校内对接 → 校外对接 → 跟进闭环 → 完成 |
| 制度建设 | 起草中 → 征求意见 → 已发布 → 完成 |

**操作：** 新建事项 → 填写标题/负责人/截止日期/说明 → 勾选清单 → 更新状态。

### 9.3 成果与展示

| 模块 | 用途 |
|------|------|
| 荣誉库 | 上传证书、专利、论文扫描件、相关 PPT |
| 照片库 | 实验室 / 设备 / 实验类图片 |
| PPT制作 | 拼装/生成汇报用 PPT |

## 12. 业务规则摘要

### 动物申领

- 空白鼠不可申领  
- 已被他人申领 / 已有待审申领 → 不可再申  
- 审核通过后写入申领人信息  

### 仪器预约

- 时长限制在仪器设定的最短–最长之间，且全局落在 **0.5–24 小时**  
- 维护中、退役不可约  
- 需培训且未完成培训（非仪器管理员）不可约  
- 时段冲突由系统拦截  

### 权限原则

- 菜单与首页按角色裁剪  
- 学生申请中心默认只看本人数据  
- 管理操作（新增仪器、导入、删除动物等）需对应管理角色

## 13. 常见问题

**Q：为什么选不了仪器时段？**  
A：检查是否维护中/退役；是否需培训且未登记培训；时长是否满足该仪器规则。

**Q：为什么列表里没有某只小鼠？**  
A：学生申领页只显示可申领个体。空白鼠、已申领、待审中、死亡不会出现。

**Q：导入仪器后没有图片？**  
A：CSV 首期只导入字段，请在仪器编辑页再上传图片。

**Q：科研助理日历上的小点是什么？**  
A：表示该日有事项截止日期，或事项在该日有更新；点击可查看并跳转模块。

**Q：如何开通新账号？**  
A：请总管理员在「用户管理」中创建并分配角色。平台关闭自行注册。
