# 数据导入模板说明

本目录提供可直接填写的 CSV 模板，供管理员批量导入。  
同学若持有 Excel 数据，请先按模板整理列名与取值，再另存为 **CSV UTF-8**，交给有权限的人导入。

| 模板文件 | 用途 | 谁可以导入 | 平台入口 |
|----------|------|------------|----------|
| [小鼠批量导入模板.csv](./小鼠批量导入模板.csv) | 代管小鼠入库 | **全体登录用户**（含学生） | 代管动物 → 批量上传；笼位总览亦可 |
| [仪器批量导入模板.csv](./仪器批量导入模板.csv) | 仪器台账入库 | **全体登录用户**（含学生） | 仪器管理 → 导入 CSV |

---

## 通用格式要求

1. **文件类型**：`.csv`（不要直接上传未转换的 `.xlsx`）  
2. **编码**：UTF-8（Excel：另存为「CSV UTF-8（逗号分隔）」）  
3. **首行必须是表头**，与模板列名一致（可删可选列，不可改必填列名）  
4. **不要合并单元格**；一行动物/一台仪器  
5. 模板中的示例行可删除，换成真实数据

## 一、小鼠批量导入

### 必填

| 列名 | 说明 |
|------|------|
| `id` | 小鼠编号，全局唯一。已存在则该行失败 |

### 推荐填写

| 列名 | 允许值 / 说明 |
|------|----------------|
| `性别` | `雄` / `雌`（也可用 male / female） |
| `品系` | 文本，如 `C57BL/6J`；空则记为「未知」 |
| `基因型` | 文本；空则「未知」 |
| `用途` | **空白** / **信号** / **免疫** / **繁殖**（也可用 blank / signal / immunity / breeding） |
| `笼位` | 笼位号（如 `A01`）或系统笼位 ID；没有则填 **`未分配`** |
| `申领人` | 平台用户姓名/邮箱/ID；没有则 **`未分配`** |
| `技术员` | 同上；没有则 **`未分配`** |
| `出生日期` | `YYYY-MM-DD`，如 `2026-04-01` |
| `进笼时间` | 可选。`YYYY-MM-DD` 或 ISO，如 `2026-06-01T10:00:00.000Z` |
| `植入时间` | 可选。植入电极时间，格式同上 |
| `采集时间` | 可选。本次/最近一次采集开始时间 |
| `上次采集时间` | 可选。上一次采集时间（用于跟踪间隔） |

英文列名亦可：`birthDate`、`cageEntryAt`、`implantAt`、`collectionAt`、`lastCollectionAt`。

### 注意

- **空白用途** 的小鼠导入后，同学端仍不可申领（需先改用途）。  
- 申领人/技术员若写成姓名，需与系统用户姓名一致才能自动绑定账号。

## 二、仪器批量导入

### 必填

| 列名 | 说明 |
|------|------|
| `名称` | 仪器中文名（也可用列名 `name`） |

### 推荐填写

| 列名 | 允许值 / 说明 |
|------|----------------|
| `英文名` | 英文名称 |
| `型号` | 型号文本 |
| `位置` | 放置地点 |
| `状态` | **可用** / **维护中** / **退役**（或 available / maintenance / retired） |
| `电话` | 默认联系电话 |
| `需培训` | **是** / **否**（或 1 / true / yes） |
| `最短时长` | 小时，**0.5–24**，建议 0.5 的倍数 |
| `最长时长` | 同上，且 ≥ 最短时长 |
| `预计修好` | 仅维护中有用，ISO 时间 |
| `维护说明` | 仅维护中有用 |
| `标签` | 多个用 `;` 或 `,` 分隔 |
| `描述` | 简介 |
| `联系人` | 默认联系人姓名 |
| `approvalName` / `approvalPhone` | 预约审批人 |
| `trainingName` / `trainingPhone` | 培训联系人 |
| `operationsName` / `operationsPhone` | 日常运维 |
| `repairName` / `repairPhone` | 维修对接 |

### 注意

- CSV **不能带图片**；导入后在仪器编辑页上传图片。  
- 导入人会成为默认仪器联系账号（`contactUserId`）。  

---

## 同学侧协作流程（推荐）

1. 下载对应模板，用 Excel 打开并填写（保留表头）。  
2. **另存为 CSV UTF-8**。  
3. 在平台 **代管动物** 或 **仪器管理** 页点击「批量上传 / 导入 CSV」，粘贴内容后提交。  
   - 学生上传小鼠：用途非空白且申领人为「未分配」时，自动归到本人名下。  
   - 学生也可导入仪器台账（导入后可在列表中看到；图片请事后由管理员或有权限者补充）。  
4. 导入结果中的错误行按提示改表后再导。
